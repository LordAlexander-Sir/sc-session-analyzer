Drop Telemetry Package
Drop .log files from Star Citizen, or a shared export .json from another pilot  ·  Also drop mission_pools.json to enable the Blueprint Pools tracker
1
Open your log folder
Press Win + R, paste this path and hit Enter:

C:\Program Files\Roberts Space Industries\StarCitizen\LIVE\logbackups
2
Select all log files
Click anywhere inside the folder window, then press Ctrl + A to select every file.

You can load files from multiple patches at once — the analyzer handles all of them.
⚑ Blueprint events only appear in 4.7+ logs
3
Drop them here
Drag the selected files directly onto this page, or use the Browse button below.

Works with 1 file or hundreds at once — duplicates are automatically skipped.
